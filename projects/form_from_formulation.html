Form from Formulation

Spring 2017


Language: P5js Processing

Code

Overview

Form from Formulation is course project for "Nature of Code" class (by Professor Moon, I audited this class at IMA, NYU Shanghai); focusing on the possibility of generating form from mathematical formulations.

This project implements parametric rose pattern.Click the open control button on the top right can access to parameters available to define this mathematical formula driven pattern.

Documentation

Rose Pattern has multiple parameters, which have different influence on the shape.

Future Development

More formulations will be explored in the future. And also more research will be placed into 3 dimensional surfaces and forms.
