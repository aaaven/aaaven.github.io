Aves

Spring 2015


Language: Processing Grasshopper

Download Code

Aves is a software and tool I build (mainly) for architects who use grasshopper and have the needs to implement a flocking system. This project conceptualizes from my experience as an architecture student doing Mercedes Benz 4S shop design project. I had the idea of generating the architecture plan layout with a human crowd simulation but I could not find any satisfying method I can achieve with no programing knowledge.
